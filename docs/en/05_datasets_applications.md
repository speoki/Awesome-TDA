# Datasets & Applications

This page collects TDA datasets and application cases.

## TDA Benchmark Datasets

### tadasets (scikit-tda)
- **Link**: https://scikit-tda.org/
- **Description**: Collection of TDA benchmark datasets
- **Content**: Standard topological shapes and test data

### MANTRA
- **Link**: https://github.com/aidos-lab/mantra-benchmarks
- **Data Repository**: https://github.com/aidos-lab/mantra
- **Full Name**: Manifold Triangulations Assemblage
- **Paper**: [MANTRA: The Manifold Triangulations Assemblage](https://openreview.net/forum?id=X6y5CC44HM)
- **Conference**: ICLR 2025
- **Description**: Manifold triangulation benchmark dataset
- **Content**: Includes data loader and benchmark code
- **Application**: Manifold learning, geometric deep learning benchmarks

### kepler-mapper Applications
- **Link**: https://kepler-mapper.scikit-tda.org/en/latest/applications.html
- **Description**: Various application cases of the Mapper algorithm

## Biomedical Applications

### Human Body Shapes Anomaly Detection
- **Link**: https://github.com/PhilippeMeyer68/body-TDA
- **Description**: Human body shape anomaly detection and classification using persistent homology

### TopoDockQ-Feature
- **Link**: https://github.com/wangru25/TopoDockQ-Feature
- **GitHub Tutorial**: https://github.com/XDaiNYU/TopoDockQ
- **Description**: Laplacian-based persistent combinatorial features for predicting protein-peptide binding quality
- **Method**: Topological deep learning model (TDL)

### PersistentSheafLaplacian_PPI_Analysis
- **Link**: https://github.com/seanfcottrell/PersistentSheafLaplacian_PPI_Analysis
- **Description**: Protein-protein interaction analysis
- **Applications**: 
  - Biomarker discovery
  - Alzheimer's disease (AD) targets
  - Drug repurposing (cross-referenced with DrugBank)
  - ADMET Lab3.0 toxicity analysis
