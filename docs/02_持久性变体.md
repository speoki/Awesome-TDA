# 持久性方法与变体

本页面介绍各种持久同调的变体和扩展方法。

## 扩展持久同调 (Extended Persistence)

### Extended Persistent Homology (EPH)
- **教程**: https://github.com/GUDHI/TDA-tutorial/blob/master/Tuto-GUDHI-extended-persistence.ipynb
- **描述**: 扩展持久同调可以捕捉超越标准持久同调的拓扑信息
- **应用**: 函数拓扑分析、地形分析

### Extended Persistence Diagrams (EPD)
- **特点**: 包含普通持久性和相对持久性信息
- **用途**: 更完整的拓扑特征描述

### DC-Filtrations
- **链接**: https://github.com/aidos-lab/diffusion-condensation-filtrations
- **全称**: Diffusion Condensation Filtrations
- **描述**: 基于扩散凝聚的过滤方法
- **应用**: 图和点云数据的拓扑分析

## Zigzag持久性

### zzgril - Quasi Zigzag Persistence
- **链接**: https://github.com/TDA-Jyamiti/zzgril
- **论文**: Quasi Zigzag Persistence: A Topological Framework for Analyzing Time-Varying Data
- **描述**: 用于分析时变数据的拓扑框架
- **应用**: 动态数据分析

### FastZigzag
- **链接**: https://github.com/taohou01/fzz
- **论文**: Fast Computation of Zigzag Persistence
- **描述**: Zigzag持久性的快速计算

### Vineyard
- **链接**: https://bitbucket.org/ahickok/vineyard/src/main/
- **另一个实现**: https://github.com/oliverchubet/persistence
- **描述**: 用于持久同调的时间序列分析
- **应用**: 功能磁共振成像(fMRI)、动态数据

### Wayne State TDA
- **链接**: https://github.com/regalski/Wayne-State-TDA
- **描述**: Vineyard生成代码，用于持久同调的时间序列分析
- **重点**: 识别随时间保持稳定的持久同调特征
- **应用**: fMRI数据分析

## 多参数持久性

### RIVET
- **链接**: https://github.com/aj-alonso/rivet
- **描述**: 用于可视化和分析双参数持久同调的程序
- **特点**: 交互式可视化工具

### GRIL - 双参数持久层
- **链接**: https://github.com/TDA-Jyamiti/GRIL
- **描述**: 基于学习的双参数持久层

### D-GRIL
- **链接**: https://github.com/TDA-Jyamiti/d-gril
- **论文**: D-Gril: End-to-End Topological Learning with 2-parameter Persistence
- **描述**: 使用双参数持久性的端到端拓扑学习

### multipers
- **链接**: https://github.com/conda-forge/multipers-feedstock
- **描述**: 多参数持久性计算

### 持久模块复形算法
- **链接**: https://github.com/TDA-Jyamiti/Algos-cplxs-pers-modules
- **论文**: Efficient Algorithms for Complexes of Persistence Modules with Applications
- **描述**: 持久模块复形的高效算法

### pyrivet (rivet-python)
- **链接**: https://github.com/rivetTDA/rivet-python/
- **描述**: RIVET的Python API
- **功能**: 
  - 计算和可视化双参数持久同调
  - 在Python脚本中调用RIVET核心引擎
  - 支持匹配距离和瓶颈距离计算
- **依赖**: 
  - 需要`rivet_console`（来自RIVET）
  - 可选：Hera的`bottleneck_dist`（用于瓶颈距离）
- **应用**: 双参数持久性的程序化计算

### Delaunay Bifiltrations (multipers)
- **链接**: https://github.com/DavidLapous/multipers
- **描述**: multipers库中的Delaunay双过滤（Delaunay Bifiltrations）功能
- **方法**: 在点云上利用Delaunay复形构建双过滤
- **优势**: 
  - 作为Rips-Density双过滤的优秀替代
  - 提供函数Rips双过滤的替代方案
- **Delaunay特性**:
  - 空圆性：三角面片外接圆内无其他点
  - 最大化最小角：避免"尖锐和细长"的三角形
  - 与Voronoi图互为对偶图
- **应用**: 
  - 点云几何数据的双参数分析
  - 多参数持久性计算
  - 三维重建和点云网格化

## 持久路径同调 (Directed Networks)

### Path Homology Toolkit
- **链接**: https://github.com/max-nekrashevich/path-homology-toolkit
- **描述**: 计算路径同调和基本路径操作的工具包

### pypph
- **链接**: https://github.com/samirchowdhury/pypph
- **论文**: Persistent Path Homology of Directed Networks (Samir Chowdhury, Facundo Mémoli)
- **语言**: Matlab, Python, C++
- **描述**: 计算网络上的持久路径同调
- **应用**: 非对称有向图分析

### persistent_path_homology-python3
- **链接**: https://github.com/rafaelpcarneiro/persistent_path_homology-python3
- **基于论文**: Persistent Path Homology of Directed Networks
- **描述**: Python实现的持久路径同调算法

### sheaves/path_homology
- **链接**: https://github.com/sheaves/path_homology/tree/main
- **描述**: 有向图的路径同调

### PathHom
- **链接**: https://github.com/WeilabMSU/PathHom
- **描述**: 路径同调计算工具

### PersNet
- **链接**: https://github.com/fmemoli/PersNet

- **语言**: Matlab
- **文档**: 包含PDF教程，需要javaPlex

## 持久拉普拉斯算子

### Persistent Laplacian
- **链接**: https://github.com/ndag/Persistent-Laplacian
- **描述**: 第q个持久拉普拉斯算子的Matlab实现
- **方法**: 通过Schur补集表述

### HERMES
- **链接**: https://github.com/wangru25/HERMES
- **描述**: 同时进行拓扑数据分析（持久Betti数）和几何数据分析（持久特征值）
- **实现**: 通过持续谱图理论
- **支持**: Alpha复形和Vietoris-Rips复形

### Persistent Sheaf Laplacians
- **链接**: https://github.com/weixiaoqimath/persistent_sheaf_Laplacians
- **描述**: 持久层拉普拉斯算子

### PersistentSheafLaplacian_PPI_Analysis
- **链接**: https://github.com/seanfcottrell/PersistentSheafLaplacian_PPI_Analysis
- **描述**: 使用持久层拉普拉斯算子进行蛋白质-蛋白质相互作用分析
- **应用**: 生物标志物发现、药物再利用

### Persistent Directed Flag Laplacian
- **链接**: https://github.com/yangzhonghuan/Persistent-Laplacian
- **描述**: 持久有向旗帜拉普拉斯算子

### PDFL-Score
- **链接**: https://github.com/mzia-s/PDFL-Score
- **论文**: Persistent Directed Flag Laplacian (PDFL)-Based Machine Learning for Protein-Ligand Binding Affinity Prediction
- **描述**: 基于PDFL的蛋白质-配体结合亲和力预测机器学习
- **依赖**: flagser包
- **数据集**: PDBbind数据集

### Persistence-Mayer-Homology-and-Laplacian
- **链接**: https://github.com/WeilabMSU/Persistence-Mayer-Homology-and-Laplacian
- **描述**: 持久性Mayer同调与拉普拉斯算子

### persistentlaplaciandatascience
- **链接**: https://github.com/tomogwen/persistentlaplaciandatascience
- **论文**: The Persistent Laplacian for Data Science: Evaluating Higher-Order Persistent Spectral Representations of Data
- **会议**: ICML 2023

### 综述论文
- **链接**: https://arxiv.org/pdf/2312.07563
- **标题**: Persistent Topological Laplacians – a Survey
