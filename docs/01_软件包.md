# 软件包与库

本页面收集了主要的TDA软件包、库和工具。

## 综合性TDA库

### giotto-tda
- **链接**: https://github.com/giotto-ai/giotto-tda
- **语言**: Python
- **特点**: 专为机器学习设计的TDA库，与scikit-learn无缝集成
- **功能**: 持久同调、Mapper、时间序列分析、图分析

### giotto-ph
- **链接**: https://github.com/giotto-ai/giotto-ph
- **语言**: Python/C++
- **描述**: Vietoris-Rips持久性的高性能CPU实现，采用GNU AGPLv3许可证
- **特点**: 
  - 改进的"无锁Ripser"实现
  - 并行化的表观对优化
  - 改进的GUDHI边缘坍缩（EC）算法
  - 支持加权VR过滤

### GUDHI
- **链接**: https://gudhi.inria.fr/
- **语言**: C++/Python
- **描述**: 由INRIA开发的综合性几何理解库
- **功能**: 
  - 多种简单复形构造
  - 持久同调计算
  - Rips、Alpha、Čech复形
  - 扩展持久同调
- **教程**: https://github.com/GUDHI/TDA-tutorial

### scikit-tda
- **链接**: https://scikit-tda.org/
- **描述**: Python TDA生态系统
- **包含组件**:
  - **ripser.py**: 快速持久同调计算
  - **persim**: 持久图距离和可视化
  - **kepler-mapper**: Mapper算法实现
  - **cechmate**: 自定义过滤构造函数
  - **tadasets**: TDA基准数据集

#### ripser.py
- **描述**: 基于Ripser的轻量级Python持久同调库
- **特点**: 简洁高效的API

#### persim
- **描述**: 持久图的距离度量和表示
- **功能**: 瓶颈距离、Wasserstein距离、持久图可视化

#### kepler-mapper
- **描述**: 灵活的Python Mapper算法实现
- **用途**: 拓扑数据可视化和探索性分析

#### cechmate
- **描述**: Python的自定义过滤构造函数
- **支持**: Alpha、Rips、Čech过滤

#### tadasets
- **描述**: 用于TDA基准测试和探索的合成数据集
- **内容**: 标准拓扑形状（环、球、簇等）

### Dionysus 2
- **链接**: https://www.mrzv.org/software/dionysus2/
- **语言**: C++/Python
- **描述**: 老牌经典TDA库，专注于同调计算和Vineyard算法
- **特点**: 
  - 时变数据分析（Vinery算法）
  - 高维持久同调
  - 成熟稳定的实现

### Eirene
- **语言**: Julia
- **描述**: 专注于高维同调计算
- **特点**: 擅长处理复杂的拓扑特征
- **优势**: Julia语言的高性能

## 高效持久同调计算

### Ripser
- **链接**: https://github.com/Ripser/ripser
- **语言**: C++
- **特点**: 世界上最快的持久同调计算工具之一
- **Python绑定**: ripserr (R), ripser.py (Python)

### Dory
- **链接**: https://github.com/nihcompmed/Dory
- **Python包**: pyDory (`pip install pydory`)
- **描述**: 大型数据集的VR持久同调和紧密环代表计算
- **输出**: H0/H1/H2持久性对文件
- **要求**: gcc-10.2 with openMP
- **应用**: 大规模点云拓扑分析

### DIPHA
- **链接**: https://github.com/DIPHA/dipha
- **全称**: Distributed Persistent Homology Algorithm
- **语言**: C++
- **描述**: 分布式持久同调计算，支持MPI集群
- **输入**: 灰度图像、距离矩阵、加权复形
- **特点**: 并行计算、对偶化优化、PHAT数据结构
- **应用**: 大规模分布式拓扑计算

### Aleph
- **链接**: https://github.com/Pseudomanifold/Aleph
- **语言**: C++
- **描述**: 用于探索和扩展持久同调应用的库
- **设计灵感**: DIPHA和PHAT
- **应用论文**:
  - 团群社群持久性
  - 二维骨骼演化分析
  - 持久化对的层级结构

### dowker_homology
- **链接**: https://sparse-dowker-nerves.readthedocs.io/
- **文档**: https://sparse-dowker-nerves.readthedocs.io/en/latest/
- **描述**: 稀疏Dowker神经和持久同调计算
- **论文**: Sparse Dowker Nerves (2018), Sparse Filtered Nerves (2018)
- **功能**: 点云、加权网络、过滤覆盖的持久同调
- **特点**: Čech复形、Alpha复形、子空间稀疏化

### PHAT（c++）
- **特点**: 为边界矩阵的持久性缩减算法提供简洁接口
- **相关**: GUDHI的边缘坍缩实现

### Sophia
- **链接**: https://bitbucket.org/schreiberh/sophia/
- **描述**: 塔（tower）的持久同调流式算法
- **核心**: 将塔转换为等价过滤，使用coning策略
- **特点**: 空间复杂度不依赖塔长度，仅依赖最大子复形
- **许可**: GNU LGPL

### CGAL
- **链接**: https://www.cgal.org/
- **语言**: C++
- **描述**: 开源计算几何算法库
- **功能**: 三角剖分、Voronoi图、凸包、Alpha shapes、网格生成等
- **应用**: GIS、CAD、分子生物学、医学影像、机器人
- **许可**: GPL v3+ / 商业许可

### OAT
- **链接**: https://openappliedtopology.github.io/
- **全称**: Open Applied Topology
- **语言**: Rust + Python
- **描述**: 通用应用拓扑软件库
- **核心**: 专为拓扑设计的高性能线性代数工具
- **特点**: 
  - 提供完整分析流程和底层构建模块
  - 开放线性代数接口供用户自定义
  - 支持持久同调计算和可视化
- **文档**: Python和Rust双语言支持

## 深度学习与TDA集成

将拓扑特征作为神经网络的输入，或设计拓扑正则化层。

### TopoModelX
- **链接**: https://github.com/pyt-team/TopoModelX
- **描述**: 拓扑深度学习库
- **支持**:
  - 单纯复形神经网络（SCNN）
  - 元胞复形网络（CCNN）
  - 其他拓扑域的神经网络
- **类似**: PyTorch Geometric (PyG)

### PersLay
- **链接**: https://github.com/MathieuCarriere/perslay
- **替代实现**: https://github.com/rballeba/perslay-pytorch
- **描述**: 将持久图向量化为神经网络层的通用框架
- **特点**: 可学习的持久图表示

### pytorch-topological
- **链接**: https://github.com/aidos-lab/pytorch-topological
- **描述**: PyTorch的拓扑层实现
- **功能**: 可微分的拓扑损失函数

### torchph
- **链接**: https://github.com/c-hofer/torchph
- **描述**: PyTorch的持久同调计算
- **特点**: GPU加速、自动微分支持

### torch-tda
- **链接**: https://github.com/CompTop/torch-tda
- **描述**: PyTorch的TDA扩展
- **功能**: 可微分的TDA层

### Topology Layer (topolayer)
- **链接**: https://github.com/bruel-gabrielsson/topolayer
- **描述**: 可微分的拓扑层
- **特点**: 允许反向传播优化拓扑损失

### Filtration Learning
- **链接**: https://github.com/git-westriver/FiltrationLearningForPointClouds
- **描述**: 学习最佳过滤参数
- **应用**: 点云数据的自适应过滤

### GUDHI (TensorFlow支持)
- **链接**: https://gudhi.inria.fr
- **描述**: GUDHI的可微分扩展，工业级标准
- **模块**:
  - `gudhi.tensorflow`: TensorFlow持久性层
  - `gudhi.wasserstein`: 可微Wasserstein距离
- **特点**: 性能稳定，生产环境可用

### TopologyLayer
- **链接**: https://github.com/bruel-gabrielsson/TopologyLayer
- **论文**: [Topology Layer for Machine Learning](https://arxiv.org/abs/1905.12200)
- **描述**: PyTorch的可微分拓扑层，经典之作
- **核心功能**:
  - 提供LevelSetLayer和RipsLayer
  - 支持0维和1维同调的可微计算
  - 实现持久同调（Persistent Homology）计算
- **应用**: 
  - 拓扑正则化（作为Loss函数中的正则项）
  - 通过反向传播优化拓扑约束
  - 将拓扑特征集成到深度学习模型




### TorchTDA
- **链接**: 
        - https://github.com/CompTop/torch-tda
        - https://zread.ai/CompTop/torch-tda
- **描述**: 轻量级TDA封装
- **功能**: 基础可微层和拓扑损失函数
- **注意**: 存在多个同名库，需甄别

### DECT
- **链接**: https://github.com/aidos-lab/dect-evaluation
- **论文**: [Differentiable Euler Characteristic Transforms for Shape Classification](https://openreview.net/forum?id=MO632iPq3I)
- **会议**: ICLR 2024
- **全称**: Differentiable Euler Characteristic Transform（可微欧拉示性数变换）
- **描述**: 端到端学习欧拉示性数变换的计算层
- **核心特点**:
  - 结合几何和拓扑特性
  - 快速且计算高效
  - 可学习任务特定表示
  - 提供与复杂拓扑深度学习层相同的拓扑表达性
- **应用**: 
  - 图分类
  - 点云分类
  - 形状分类
- **优势**: 性能与更复杂模型相当，但更简洁高效

### Topological Uncertainty
- **链接**: https://github.com/tlacombe/topologicalUncertainty/
- **论文**: Topological Uncertainty: Monitoring Trained Neural Networks through Persistence of Activation Graphs
- **描述**: 通过激活图持久性监测训练过的神经网络
- **核心思想**: 
  - 从加权图中提取拓扑信息
  - 通过Fréchet均值聚合拓扑信息
  - 计算拓扑距离检测异常
- **应用**: 
  - 分布外（OOD）样本检测
  - 神经网络异常监测
  - 权重分布异常检测
- **当前支持**: TensorFlow 2顺序网络、全连接层
- **未来**: 将集成到GUDHI库，支持PyTorch
